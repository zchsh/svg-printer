Demo - Paste & print SVGs on a letter page

Paste SVGs into the page, reposition them, then print the page. Most browsers will let you save a PDF document from the print dialog, which you can then print later.


FAQ

I made this tool to work with a Paper miniature generator that outputs SVGs.

Caveats and shortcomings This tool is far from perfect. Please keep in mind the following caveats:
  • If you're pasting different SVGs, you might see strange shapes and images where they shouldn't be. This is a bit technical, but pasted SVGs all share the same parent document, so any <defs /> or other elements with the same id will cause conflicts. There's a fix for this, but so far I've been too lazy to implement it... please email me at hi@zch.sh if you run into it, then I'll know it's worth the effort.
